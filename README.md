## El local de comida actualmente no cuenta con una plataforma digital propia que permita mostrar sus productos, promociones y datos relevantes de manera clara y accesible.
Esto genera dificultades para los clientes al momento de conocer la oferta disponible, realizar pedidos de forma rápida y confiable, y obtener información como ubicación y horarios.
Además, la ausencia de un sistema digital limita las oportunidades de aumentar ventas, mejorar la experiencia del cliente y fortalecer la presencia del negocio en internet.
## Desarrollar un sistema web para el local de comida que permita exhibir sus productos, facilitar la realización de pedidos online y mejorar la interacción con los clientes, contribuyendo al incremento de ventas y posicionamiento digital del negocio.
## Objetivos específicos
Diseñar una interfaz web intuitiva que permita visualizar el catálogo de productos, promociones y datos del local de forma clara y atractiva.
Implementar un sistema básico de pedidos online con carrito de compras para simplificar el proceso de compra.
Integrar herramientas de comunicación directa (como WhatsApp) y redes sociales para mejorar la atención al cliente y la difusión del negocio.
## Beneficios
Mayor visibilidad del negocio en internet.
Incremento en las ventas al facilitar el proceso de compra.
Mejora en la experiencia del usuario (todo más rápido y claro).
Automatización básica de pedidos.
Generación de confianza mediante opiniones de clientes.
Acceso rápido a información importante (ubicación, horarios, contacto).
## Alcance del proyecto
El sistema web incluirá:
Página principal con información del local (historia, identidad, imágenes).
Catálogo de productos con precios (tipo menú online).
Sección de promociones o combos.
Carrito de compras básico.
Sistema de pedidos online.
Botón de contacto directo a WhatsApp.
Visualización de ubicación y horarios.
Sección de opiniones de clientes.
Integración con redes sociales.
No se incluirán en esta etapa:
Sistema de pagos online (solo pedidos).
Gestión avanzada de stock.
Panel administrativo complejo (puede plantearse como mejora futura).
